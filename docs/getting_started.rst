Getting Started
===============

This page details how to get started with molecool. 

Installation
------------

Dependencies: 
*************
#. Python 3.7+   
#. Matplotlib  
#. Pandas  

To do a development install use::

    pip install -e .

Usage
------
once installed, you can use the pkg::

    import molecool 
    benzene_symbols, benzene_coords = molecool.open_xyz(benzene.xyz)
